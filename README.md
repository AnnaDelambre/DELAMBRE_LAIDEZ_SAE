Ce dépôt contient les éléments principaux du projet d’analyse des valeurs foncières.

Vous y trouverez :
- une capture d’écran de la datavisualisation réalisée sur Power BI ;
- les scripts SQL de création des tables Oracle (via SQL Developer), disponibles dans le dossier BASE_DE_DONNEES ;
- les pipelines et workflows Apache Hop dans le dossier ETL. Ce dossier ETL inclut également un fichier workflow.bat qui permet d’automatiser l’exécution des traitements. Ce script lance les workflows nécessaires pour alimenter le schéma en étoile à partir des données sources.

L’ensemble du dépôt permet donc de reconstruire la base de données, exécuter les traitements ETL et visualiser les résultats via le tableau de bord.
